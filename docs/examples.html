EXAMPLES


The Matrix Dex is the first dex. Workflow to make this dex is:

1. Edit data.json

2. 

python run.py

3. Add updated HTML files from gen/ folder to the project hosted on GitHub Pages. https://alinoorul.github.io is the free URL thanks to GitHub.

Dexs by dexweb

1. The Matrix Dex
The original made by Noorul Ali to put his projects on the Internet. Dexweb was originally just a Python script to generate this website from fast additions and edits in data.json.

2. Dexweb Docs
Documentation for Python library dexweb. Yes, this documentation website is a dex made by dexweb. Meta, we know!

3. Protocol Dex
This dex edits 'index_template' in config.json to implement search functionality by adding a < script > tag for JavaScript, embeds images in webpages (see advanced dexweb usage) using < img > tags, and uses < table >, < ol > and < ul> tags. These tags are used directly in paragraph strings.

4. Islahe Nafs
This dex is generated from a book Learn To Swim Before You Sink by Dr. Mohsin Ali. This dex implements advanced functionality using JavaScript. A random page from data.json is displayed on the index page. It captures the user experience of opening the book randomly on a page to find guidance relevant to the user.
